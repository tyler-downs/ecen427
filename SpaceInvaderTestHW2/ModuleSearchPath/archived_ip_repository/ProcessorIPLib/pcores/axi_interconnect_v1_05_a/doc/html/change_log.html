Change log for AXI Interconnect

Changes in v1.05.a, introduced in 13.4

Note: Unless specified, limitations and resolved issues affect all previous versions.

13.4 - Changes in VHDL/Verilog/Netlist sources (.vhd, .v, .ngc, .edn)

 
New Features:

  • Initial release of v1.05.


Resolved issues:

  • Improved support for masters (SI slots) with wide thread-ID signals -- reduced utilization and path delays. Also, ACCEPTANCE and ISSUING limits support integer range 1-32.


Known Issues / Limitations:

  • Setting parameter C_S_AXI_IS_INTERCONNECT = 1 is not yet supported.

  • The diagnostic control interface (S_AXI_CTRL) is not implemented. The parameter C_USE_CTRL_PORT is therefore forced to 0.
